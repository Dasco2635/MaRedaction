# Virtualisation et optimisation des ressources avec Windows Server

Ce dépôt contient un article technique sur la virtualisation avec Windows Server, et plus particulièrement l’utilisation de Hyper-V pour optimiser les ressources dans un environnement professionnel.

## Contenu
- Introduction à la virtualisation
- Fonctionnement de Hyper-V
- Préparation de l’infrastructure
- Optimisation des ressources (mémoire, stockage, CPU)
- Sécurisation des environnements virtualisés
- Automatisation et gestion avancée

## Objectif
Ce projet a été réalisé dans le cadre d’une évaluation sur l’utilisation de Git et GitHub. Il met en avant la structuration d’un article technique, le suivi des tâches via un Kanban board, et la gestion collaborative avec des branches Git.
